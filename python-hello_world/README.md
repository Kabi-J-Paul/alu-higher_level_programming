# Python - Hello, World

The first Python project in the ALU higher-level programming.
It introduces the basics of writing and running Python scripts, printing
text and variables, and working with strings through indexing and slicing.

## Learning Objectives

- How to run a Python script and run Python code inline from the shell
- How to print text and variables using `print`
- How to use strings, indexing, and slicing
- How to format output with f-strings
- The official Python coding style and how to check it with `pycodestyle`

## Requirements

- Ubuntu 20.04 LTS, `python3` (version 3.8.5)
- Python files begin with `#!/usr/bin/python3`; shell scripts with `#!/bin/bash`
- All files end with a new line and are executable
