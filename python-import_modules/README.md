# Python - import & modules

The third Python project in the ALU higher-level programming.
It covers how to split code across files and reuse it: importing functions
and variables from modules, controlling what runs when a file is imported
versus executed, and reading command-line arguments.

## Learning Objectives

- Why Python programming is awesome
- How to import functions from another file
- How to use imported functions
- How to create a module
- How to use the built-in function `dir()`
- How to prevent code in a script from being executed when imported
- How to use command line arguments with a Python program

## Requirements

- Ubuntu 20.04 LTS, `python3` (version 3.8.5)
- Every file begins with `#!/usr/bin/python3` and ends with a new line
- All files are executable
- Code follows `pycodestyle` (version 2.7.*)
- File length is tested with `wc`
- You are not allowed to use `*` for importing or `__import__`
